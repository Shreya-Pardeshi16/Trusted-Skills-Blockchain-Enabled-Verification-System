# Trusted Skills – Blockchain-Enabled Verification System

Trusted Skills is a secure digital credential verification platform designed to prevent fake certificates and simplify academic document verification using blockchain-inspired architecture, OCR validation, encryption, digital signatures, and QR-based authentication. The system allows students to upload certificates, institutions to verify and digitally sign them, and employers to instantly validate credentials through QR code scanning. Verified documents are securely stored using AES encryption and SHA-256 hashing to ensure confidentiality, integrity, and tamper resistance.

### Key Features

* OCR-based automatic document validation
* Blockchain-style immutable transaction records
* AES encryption for secure document storage
* SHA-256 hashing for tamper detection
* Digital signature verification by institutes
* QR-based instant employer verification
* Skill Wallet for managing verified certificates
* Role-based modules for Admin, Student, Institute, and Employer
* Dual blockchain server architecture for redundancy and reliability
* Fast, transparent, and secure credential sharing system

### Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Spring Boot, Flask, Java, Python
* Database: MySQL
* Security: AES Encryption, SHA-256, Digital Signatures
* OCR: Azure Cognitive OCR Service
* Storage: IPFS / Distributed Storage System
